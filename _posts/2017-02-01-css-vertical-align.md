垂直居中是css中的老大难，也是身为前端的基本功，今天想结合下项目中遇到的问题，介绍一下时下流行的垂直居中方法的一些小瑕疵。

## vertival-align

>* 大家都知道vertical-align只针对与display:table 或者 inline 或者 inline-block的元素。


```
var Person = function() {
  this.canTalk = true;
};
```

>* 构造函数的prototype是指原型
>* 大家共用的方法写在构造函数的原型上
>* 给Person类写上了共用的greet方法

```
Person.prototype.greet = function() {
  if (this.canTalk) {
    console.log('Hi, I am ' + this.name);
  }
};
```
