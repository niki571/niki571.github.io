好激动作为一个程序媛，我终于有自己的博客了！！！

今年前端圈吵架都有好几波，对于技术的争论一切都是因为前端发展的太快了！还记得刚入行的时候一个iOS程序员很鄙夷地说，现在什么时代了还学web前端我教你iOS，可是两年不到的时间创业融资的泡泡灭了不少也萎靡了iOS。刚入行时候的小师父已经到请教我问题的地步，这个行业是不进则退的最大体现。

最近思考了很多自己的发展方向。想起一开始想写代码的心情，当时有几个比较好的朋友是很厉害的程序员，他们身上散发的那种geek气质让我觉得充满光芒。我喜欢的并不只是前端，我喜欢代码可以让我实现自己想法，喜欢代码可以让我和一群很聪明的人在一起。2015年机缘巧合去了某五百强，现在回过头来看这个选择和我一开始想写代码的心情有点违背。虽然说进了大公司，但是至少我所在的team让我觉得程序员都只是搬砖工。老大根本就不在乎技术更在乎的是能不能满足国外每天的需求，忙的时候一个星期release三四次。

现在的我更在乎自己两方面的发展，1在运用新技术的基础上能探究其底层究竟。有一个自己的实用性的开源项目，讲究设计不管是大至原型还是小至颜色logo都是通过自己的想法实现的。2017年希望自己可以创造而不是工作上的重复。

目前掌握的技术：

>* html css javascript
>* jquery
>* es6
>* react redux react-router
>* webpack

2017年的小目标：

>* 成为一个积极更新博客的好青年
>* 不浮于表面运用更深层次地钻研前端框架等技术的核心思想
>* 有一个自己idea产生的实用性的开源项目
